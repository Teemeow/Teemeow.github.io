Contexte

Durant ma deuxième annèes de BTS SIO j'ai réalisé un projet de jeu concours, le coucours consiste à élire la meilleur réalisation, une réalisation est un travail éffectué (repeindre une pièce ou crée un meuble...). Les votes se font sur une application mobile et l'affichage des résultats ce fait sur une application web, les deux applications s'échange des données grâce à une API.

Organisation

Pour ce projet j'était en binôme, pour organiser notre travail nous avons utilisé la méthode agile en faisant plusieurs sprints.

Nous avons utilisé Trello ainsi que GitHub pour versionner notre code

Réalisation

Nous avons commencé par créer le formulaire pour poster sa réalisation

Ensuite nous avons crée l'API pour envoyer les données des réalisations sur l'application mobile.

Sur l'application mobile on a ajouté la connexion grâce à un code puis le système de vote. Lors du vote on choisit 3 réalisations et on leur attribue chacun une note de 1 à 5. Après l'API pour exporter les données vers application web.

Sur l'application web nous ajouté le classement et pour finir nous avons implémenté le système de connexion avec différents rôles sur le site
