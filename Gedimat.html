Contexte

Durant ma deuxième annèes de BTS SIO j'ai réalisé un projet de jeu concours, le coucours consiste à élire la meilleure réalisation, une réalisation est le résultat d'un travail accompli, qu'il s'agisse de repeindre une pièce ou de créer un meuble. Les votes se font sur une application mobile et l'affichage des résultats ce fait sur une application web, les deux applications s'échange des données grâce à une API.

Organisation

Pour ce projet j'était en binôme, pour organiser notre travail nous avons utilisé la méthode agile en faisant plusieurs sprints.

Nous avons utilisé Trello ainsi que GitHub pour versionner notre code

Réalisation

Nous avons commencé par créer le formulaire pour poster sa réalisation.

Ensuite nous avons crée l'API pour envoyer les données des réalisations sur l'application mobile.

Sur l'application mobile, nous avons ajouté la fonctionnalité de connexion via un code, puis nous avons mis en place un système de vote. Lors du vote, l'utilisateur peut choisir entre trois réalisations et leur attribuer une note de 1 à 5. Ensuite, nous avons implémenté une API pour exporter les données vers l'application web.

Le projet n'est pas encore fini, il nous manque à ajouter sur l'application web l'affichage du classement, et pour finir le système de connexion avec différents rôles/dorits sur le site
