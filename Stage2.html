Stage 2er année

Contexte

Du 3 janvier au 10 février 2023 j'ai effectué mon stage dans l'entreprise Com & Company qui est une agence web qui travail avec le Groupe Valority (anciennement Valeur et Capital), groupe qui est dans le domaine immobilier. Cette entrpirse se situe à Lyon (94 Quai Charles de Gaulle). Dans cette entreprise j'ai rejoint le service informatique qui développe des applications pour le groupe, ce service est répartie en plusieurs équipe via un système d'ilôt de 4 à 7 personnes.

Missions / Tâches

Ajout de colonnes

Sur la page d'accueil de l'application Gaia, application qui sert à la gestion des locations et au suivi des mandats de gestion, je devais ajouter deux nouvelles colonnes dans un listing et de permettre l'export sous excel de ce listing.

J'ai commencé par implémenter l'affichage en ajoutant une colonne pour les gestionnaires et une pour la nature des lots (appatement, garage...).

Puis j'ai implémenté le bouton d'export Excel grâce à Laravel Excel.

J'ai principalement utilisé le Framework Laravel, complété par Bootstrap et jQuery pour la gestion l'affichage. Une fois l'évolution prête, j'ai crée une merge request afin qu'un autre membre de l'ilôt le révise. En prenant en compte ses retours, j'ai ensuite corrigé et poussé le code sur ma branche.

Ajout de champs de saisie

Le but de cette évolution sur Gaia était d'ajouter l'id d'un bail sur un document, ce document est envoyé sur Juptier (application qui permet de centraliser les données de toutes les applications) lors de l'ajout d'un état des lieux.

Pour commencer j'ai implémenté la saisie du bail dans le formulaire d'ajout d'état des lieux.

Après la saisie j'ai géré l'envoie des données dans Jupiter et dans la base de donnéés.

Ensuite j'ai crée une merge request, j'ai traité les retours puis j'ai réalisé un déploiment dans l'environement de qualif (environement de test), une fois les tests fait j'ai encore traité les retours.

Bilan

Mon stage à durée 6 semaines ce qui était court, mais j'ai eu le temps d'apprendre sur plusieurs technologies comme Laravel, Bootstrap, jQuery, j'ai aussi appris à être beucoup plus autonome.
