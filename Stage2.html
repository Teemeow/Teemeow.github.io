Stage 2er année

Contexte

Du 3 janvier au 10 février 2023 j'ai effectué mon stage dans l'entreprise Com & Company qui est une agence web qui travail avec le Groupe Valority, groupe qui est dans le domaine immobilier. Cette entrpirse se situe à Lyon (94 Quai Charles de Gaulle). Dans cette entreprise j'ai rejoint le service informatique qui développe les applications, ce service est répartie en plusieurs équipe via un systeme d'ilôt de 4 à 6 personnes.

Missions / Tâches

1erMission: Sur la page d'accueil de l'application Gaia application qui sert à la gestion des locations et au suivi des mandat de gestion, je devais ajouter deux nouvelles colonnes dans un listing et de permettre l'export sous excel de ce listing.

Technologies utilisées:

  • Laravel
  • Bootstrap
  • jQuery

J'ai commencé par implémenter l'affichage en ajoutant une colone pour les gestionnaires et une pour la nature des lots(appatement, garage...).

Puis j'ai implémenter le bouton d'export Excel grâce à Laravel Excel

J'ai utilisé le Framework Laravel, pour gérer l'affichage j'ai utilisé Bootstrap et jQuery pour le bouton. Quand j'avais fini de coder l'évolution j'ai crée une merge request pour que un membre de l'ilôt le regarde et fasse des retours dessus, j'ai ensuite traité les retours puis j'ai poussé le code sur ma branche.

2erMission:Nouvelle évolution sur Gaia qui servait à ajouter un bail sur le document envoyé sur Juptier lors d'un ajout des état des lieux. Quand un état des lieux est ajouté un document est envoyé sur une autre application qui permet de centraliser les données (Jupiter) , lors de cette evoie de document il fallait que j'ajoute l'id du bail afin de le lier au document.

Technologies utilisées:

  • Laravel

Pour commencer j'ai ajouté la saisie du bail d'un le formulaire d'ajout d'état des lieux

Après la saisie j'ai géré l'envoie des données dans Jupiter et dans la base de donnéés

Ensuite j'ai crée une merge request, j'ai traité les retours puis j'ai mis ma branche en qualif qui est une branche de test, une fois les tests fait j'ai encore traité les retours.

Bilan

Mon stage à durée 6 semaine ce qui était court mais j'ai eu le temps d'apprendre sur différentes technologies comme Laravel, Bootstrap, jQuery.
